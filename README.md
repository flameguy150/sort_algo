## How to run

1. Clone the repository
```bash
git clone https://github.com/yourusername/sort_algo.git
cd sort_algo
```

2. Install pygame
```bash
pip install pygame
```

3. Run file
```bash
windows: py sorter.py
mac: python3 sorter.py
```
